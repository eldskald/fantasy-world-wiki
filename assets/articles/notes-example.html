Notes example

This example is to showcase how you can make little blocks of text in the middle of your article. It is very common for RPG books/texts to use this type of technique in the middle of a text to talk about rules, ideas or anything related to the text but is kinda tangential to it in a sense. Remember that the articles are here to show how they look on an article, the code is on source file which, in this case, is in assets/articles/notes-example.html.

Floating notes

Floating left

Note title

Write whatever you want in here. You can put other tags as well, like the h5 I just used to give it a title, or the em tag to give emphasis to text by making it italic, like this.

Full size note

A note without either the right-float and the left-float will be a full size one. Also, on mobile all notes are full size actually. Always try your page on mobile to make sure it still looks good there, especially because your players might primarily access it from their phones.
